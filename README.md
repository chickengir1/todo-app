# Angular Todo List

Angular와 Tailwind CSS로 만든 간단한 Todo List 웹 애플리케이션입니다.

## 기능

- 할 일 추가, 완료/미완료 토글, 개별 삭제
- 완료된 할 일 일괄 삭제
- 남은 할 일 개수 표시
- 반응형 UI

## 사용 기술

- **프레임워크** Angular
- **언어** TypeScript
- **스타일링** Tailwind CSS
- **상태 관리** Angular Signals

## 실행 방법

1.  **의존성 설치**
    ```bash
    npm install
    ```
2.  **개발 서버 실행**:
    ```bash
    ng serve
    ```
